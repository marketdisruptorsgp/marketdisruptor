# Market Disruptor

Web application for Market Disruptor.

## Local development

Prerequisites:
- Node.js + npm (recommended via nvm)

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm i
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the app:

```sh
npm run build
npm run preview
```

Deploy the output using your hosting provider’s standard Vite/SPA deployment flow.
